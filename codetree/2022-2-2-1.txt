빵 구하고자 하는 사람 m명
m번 사람은 정확히 m분에 베이스캠프에서 출발하여 편의점으로 이동
출발 전엔 격자 밖에 나와있음

격자 크기 n x n

1. 
격자에 있는 사람들 모두가 가고 싶은 편의점 방향 향해 최단거리로 1칸 이동
상좌우하 우선 순위
=> 중간에 못가게 되는 경로가 생길 수 있으니 bfs, route 탐색,
이동 시 cur 업데이트

2.
편의점 도착하면 멈추고, 이때부터 다른 사람들은 해당 편의점 있는 칸 지날 수 없음
격자에 있는 사람들이 모두 이동한 뒤에 이동 불가 처리됨
=> 편의점 도착 시 board에 이동 불가 업데이트

3. 
현재 시간이 t분이고 t<=m이면, t번 사람은 가고 싶은 편의점과 가장 가까운 베이스 캠프에 들어감
행이 작은, 열이 작은 우선.
이때부터 다른 사람들은 해당 베이스캠프 칸 지날 수 없음.
=> 베이스캠프가 선정 가능한지 먼저 확인
베이스캠프 위치 board에 이동 불가 업데이트

각 사람마다 가고 싶은 편의점 위치, 베이스 캠프 위치 겹치지 않음
이동 도중 동일한 칸에 둘 이상의 사람이 위치하게 되는 경우 가능

입력
n, m
격자 정보 (0 빈칸, 1 베이스캠프)
각 사람들이 가고자 하는 편의점 위치

출력
모든 사람이 편의점에 도착하는 시간 출력

변수
board 격자 정보 저장 (0 빈칸, 1 베이스 캠프, -1 이동 불가)
camp 베이스캠프 위치 저장
store 편의점 위치 저장
cur 사람 현재 위치 저장
arrived 도착 여부 저장
