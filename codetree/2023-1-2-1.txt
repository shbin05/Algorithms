M명 참가자
미로 N x N 좌표 (1,1)로 시작
미로 구성
    1) 빈칸: 이동 가능
    2) 벽: 이동 불가, 내구도 1-9, 회전할때 내구도 1감소하여 0이 되면 빈칸 전환
    3) 출구: 도달 시 즉시 탈출

1초마다 모든 참가자 한 칸씩 움직임
최단거리: |x1-x2| + |y1-y2|
모든 참가자 동시에 움직임
출구와 가까워지는 방향으로 이동
상하좌우 이동, 상하 우선시
움직일 수 없으면 이동 x
한 칸에 2명 이상 존재 가능

참가자 이동 종료 후 미로 회전
1. 한 명 이상의 참가자와 출구를 포함한 가장 작은 정사각형을 찾음.
2. 2개 이상일 시 r좌표 작고, c 좌표 작은 것 우선.
3. 정사각형 시계방향으로 90도 회전하며, 회전된 벽은 내구도 1깎임.

K초 동안 반복
모든 참가자가 탈출에 성공하면 게임이 끝남.
모든 참가자들의 이동 거리 합과 출구 좌표 출력.

moved: 참가자 이동 거리 저장
escaped: 참가자 탈출 여부 저장

rotate(square):
    정사각형 시계방향으로 90도 회전
    값 0 초과 시 1씩 감소

find_square():
    한 변 길이 2부터 시작
    while True:
        좌상단, 우상단, 좌하단, 우하단 순으로 출구 포함한 정사각형 탐색

update_exit():
    출구 좌표 업데이트

main():
    N, M, K 입력 받음
    미로 정보 입력 받음
    참가자 좌표 입력 받음
    출구 좌표 입력 받음

    for K:
        nboard = deepcopy(board)
        for board:
            유저 좌표 발견 시:
                상하좌우 탐색하며 출구에 가까워지는지 여부 체크
                가까워질 시: 
                    이동거리 += 유저 수
                    기존 좌표 -= 유저 수
                    새 좌표가 출구가 아닐 시: nobard에 좌표 업데이트
                    새 좌표이면: 탈출 += 유저 수
                    break
        모두 탈출 시 break
        find_square()
        rotate(square)
        update_exit()
