마을 크기 N x N
도로는 0 아닌 곳은 1

메두사는 오직 도로만을 따라 최단 경로로 공원으로 이동.
집과 공원은 항상 도로위에 있음.

M명의 전사들이 메두사를 향해 최단 경로로 이동.
도로 비도로 구분 않고 이동.

1. 메두사 이동
도로를 따라 한 칸 이동.
공원까지 최단 경로로.
이동한 칸에 전사가 있을 경우 전사는 메두사에게 공격 받고 사라짐.
상하좌우 우선순위.

2. 메두사 시선
상하좌우 중 전사를 가장 많이 볼 수 있는 방향을 바라봄.
메두사의 시야각 안에 있어도 다른 전사에 가려진 경우 보이지 않음.
메두사가 본 전사들은 현재 턴에는 움직일 수 없음.
두 명 이상 전사가 같은 칸에 있다면 해당 칸 전사 모두 움직일 수 없음

3. 전사 이동
돌로 변하지 않은 전사에 한해 메두사 향해 최대 두칸 이동.
첫 이동:
    메두사와의 거리를 줄일 수 있는 방향으로 한 칸 이동.
    상하좌우 우선순위
    격자 바깥이나 메두사의 시야에 들어오는 곳으로는 이동할 수 없음.
두번째 이동:
    메두사와의 거리를 줄일 수 있는 방향으로 한 칸 이동.
    좌우상하 우선순위
    격자 바깥이나 메두사의 시야에 들어오는 곳으로는 이동할 수 없음.

4. 전사 공격
메두사와 같은 칸에 도달한 전사는 메두사 공격하고 사라짐.

입력
N,M
메두사 집 위치, 공원 위치
전사들 좌표
마을 도로 정보

출력
매 턴마다 모든 전사가 이동한 거리 합, 돌이 된 전사의 수, 메두사를 공격한 전사 수
단, 메두사가 공원에 도착하면 0 출력하고 프로그램 종료
공원에 도착할 수 없다면 -1 출력

변수
board = [] 도로 정보 저장 (0: 도로, 1: 비도로)
wars = [] 전사 좌표 저장
sight = [] 시야 저장 (매 턴 초기화)
stunned = [] 돌이 된 전사 (매 턴 초기화)
attacked = [] 공격한 전사 (매 턴 초기화)
